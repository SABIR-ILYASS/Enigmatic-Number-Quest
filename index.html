Règles du jeu :

  • Trouvez la séquence de chiffres cachée
  • 'V' : chiffre correct et bien placé
  • 'E' : chiffre correct mais mal placé
  • 'F' : chiffre absent de la séquence

Nombre cible : 7582
Votre essai : 7284
Résultat : V E V F

Help!

Rules of the game:

  • You need to find a sequence of digits
  • If a digit is correct and in the right position, 'V' is displayed
  • If a digit is correct but in the wrong position, 'E' is displayed
  • If a digit does not exist in the target number, 'F' is displayed
  • The game ends when the number is found

Example:
Target Number: 7582
Guess: 7284
Response: VEVF
(7 is correctly placed, 2 and 8 are correct but misplaced, 5 is absent)
